## Instalar proyecto

1. Renombrar el archivo .env.example a .env

```bash
mv .env.example .env
```

2. Instalar las dependencias del proyecto

```bash
npm install
```

3. Iniciar la aplicacion con docker

```bash
docker compose up -d
```
